This site is [my](https://www.skybluetrades.net/) working notebook and public
record for a pilot of a class on the use of AI tools in research that is being
run at [LAE](https://lae.mit.edu/) this summer.

The project I'm going to be working on is a look at some reduced-order models
of cloud and contrail dynamics, taking a dynamical systems perspective.

## What you can find here

- [Research log](log/): one entry per session using a coding agent (more or less).
- [Project area](project/): background, methods, artifacts, and findings.
- [Bio page](bio/): slightly whimsical view of my life so far.
