[Transcript](../../sessions/001-ssg-choice/)

- A first short session to decide on a <Wikipedia>static site
  generator</Wikipedia> framework to use, and to get the initial project set
  up.
- I use a framework called [Astro](https://astro.build/) for [my personal
  website](https://www.skybluetrades.net/), and so decided to use the same for
  this one.
- This all went much as expected, apart from a **permissions problem**. This
  happens all the time when setting agents up in a sandbox, and this was the
  first time I was using [Nono](https://nono.sh/), so that wasn't expected.
- I also made sure that the `.pi` directory doesn't end up in Git.
- The agent set up Astro [here](../../sessions/001-ssg-choice/#entry-5) following
  [my instructions](../../sessions/001-ssg-choice/#entry-3), with MDX input, KaTeX
  mathematics rendering, the fonts and overall colors I selected. I also got
  the agent to copy the overall layout (Tufte-inspired) and some components
  (margin notes mostly) from my main website.
