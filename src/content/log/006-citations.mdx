- A long session of me mostly saying "Yes" to get the agent to build a
  citation handling system.
- This was another `grill-me` session. It may look redundant to have all these
  questions that you mostly just agree to, but it's an important part of
  ensuring that the agent is heading in the right direction. All of the
  information it collects during these grilling sessions goes into the
  description of work it uses to direct itself in building the features you
  ask for. The more information it has, the more likely it is to do things
  right first time (or close to right).
- In this case, there was [one tiny problem to
  correct](../../sessions/006-citations/#entry-109), but otherwise it really did
  work first time.
